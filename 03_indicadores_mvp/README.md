# 📊 Indicadores MVP — Atlas Social de Hortolândia

**Diretório:** `03_indicadores_mvp/`  
**Objetivo:** Estruturar indicadores analíticos iniciais (MVP) capazes de responder perguntas reais da política pública de Assistência Social.

---

## 🎯 Propósito

Este diretório concentra os **primeiros indicadores operacionais do projeto**, construídos a partir da integração entre:

- Cadastro Único (CadÚnico)
- Estrutura territorial (loteamento / núcleo / RP)
- Rede de atendimento (CRAS, CREAS, OSC)

Os indicadores aqui definidos têm caráter **aplicável**, não apenas descritivo.

> **Princípio orientador:**
> Não medir apenas a vulnerabilidade — medir **onde ela está, com que intensidade, e se está sendo atendida.**

---

## 🧱 Estrutura dos Indicadores

Os indicadores seguem uma lógica comum:


Cada indicador responde a uma ou mais das seguintes dimensões:

| Dimensão | Pergunta-chave |
|----------|---------------|
| Vulnerabilidade | Onde estão as famílias em maior risco? |
| Território | Há concentração espacial da vulnerabilidade? |
| Atendimento | O Estado está chegando onde precisa? |
| Infraestrutura | Há fatores urbanos agravando a vulnerabilidade? |

---

## 📌 Indicadores Disponíveis

### 1. Densidade de Vulnerabilidade Territorial
📄 `ind_densidade_vulnerabilidade_v02.md`

Mede a intensidade da vulnerabilidade por loteamento a partir de três camadas:

- **Densidade Bruta** (famílias / domicílios)
- **Coabitação** (múltiplas famílias por domicílio)
- **Gap de Cobertura** (famílias não atendidas)

> Este indicador estrutura a base analítica do projeto.

## 🔗 Conexão com as Perguntas da Secretaria

Os indicadores deste diretório foram desenhados para responder diretamente às demandas da Secretaria de Inclusão Social:

- Concentração territorial da vulnerabilidade
- Famílias sem acompanhamento
- Relação entre território e demanda
- Cobertura da rede socioassistencial

---

## 🧠 Diretriz Metodológica

- Os indicadores são construídos sobre **dados relacionais estruturados**
- Não dependem exclusivamente de dados externos (ex: IBGE)
- São **auditáveis e reproduzíveis**
- Respeitam integralmente a LGPD (sem exposição de dados pessoais)

---

## 🔐 Política de Dados

Este diretório não contém dados brutos.

- ❌ Não contém microdados do CadÚnico
- ❌ Não contém identificadores pessoais
- ✅ Contém apenas lógica analítica e documentação

---

## 🚀 Papel no Projeto

Este diretório representa a transição de:

> **dados → informação → inteligência aplicada à gestão pública**

Os indicadores aqui definidos são a base para:
